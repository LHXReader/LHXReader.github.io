网页的生成和书写

用Hexo创建网页

  • clone 克隆库
  • 初始化网页
    • hexo init folder
    • cd folder
    • npm install
      Hexo 将会在指定文件夹中新建所需要的文件
  • git pull
    拉取
  • hexo clean
    清除缓存文件和已生成的静态文件
  • hexo n post “文件名”
    新建一篇文章
  • hexo g
    生成静态文件
  • hexo s
    启动服务器
  • hexo d
    部署网站
  • git add —all
    添加
  • git commint -m “reader”
    注释
  • git push -u origin master
    提交
